Database Connection Error

Error: Failed to connect to the database. This could be because:

  • MySQL server is not running
  • Database credentials are incorrect
  • The casino_db database has not been created
  • PHP MySQL extension is not installed

How to Fix This Issue:

  1. Install Required Software: Make sure you have installed XAMPP, WAMP, or another PHP/MySQL package.
  2. Start Services: Ensure Apache and MySQL services are running.
  3. Run Setup Script: Use the database setup script to create the necessary database and tables.
  4. Check Configuration: Verify that the database credentials in config.php are correct.
